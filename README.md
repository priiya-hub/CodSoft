# Java Mini Projects

# Project Overview 
This repository contains simple Java mini projects created to practice and strengthen core programming concepts.

## Projects included
1. Number Guessing Game- A game where the program generates a random number and the user tries to guess it.
2. Student Grade Calculator- Calculate total marks, average percentage and assign grades based on input.
3. ATM Interface- Simulates real-world ATM operations like-
 -Balance check
 -Cash Withdrawal
 -Deposit Money  

## Technologies Used-
-Java
-Loops and conditional statements
-User input using Scanner

## How to Run
1. Clone the repository
2. Open in any Java IDE
3. Compile and run the program
